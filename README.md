# gobuild

## 安装说明
#### 安装依赖

下载 [gox](https://github.com/mitchellh/gox) 

	go get github.com/mitchellh/gox
	
	# build toolchain
	gox -build-toolchain

#### 安装主程序
	go get github.com/shxsun/gobuild
	
	# $ cd到程序所在目录, 修改app.ini
	go build
	./gobuild
